# AI食安系统 — 架构文档

> 生成日期：2026-05-24 | 版本：v1.0

---

## 一、技术栈总览

| 层级 | 技术 | 说明 |
|------|------|------|
| 前端框架 | Vue 3 (Composition API + `<script setup>`) | SPA 单页应用 |
| UI 组件库 | Element Plus | 饿了么 UI 组件库 |
| 路由 | Vue Router 4 | 嵌套路由 + 懒加载 |
| 构建工具 | Vite 8 | 极速 HMR 热更新 |
| 后端框架 | Express 5 | Node.js Web 框架 |
| 数据库 | better-sqlite3 | 轻量级 SQLite（WAL 模式） |
| AI 引擎 | DeepSeek / 通义千问 | OpenAI 兼容接口，未配置时自动降级 |
| OCR 识别 | 本地 OCR 工具 | 标签图片文字提取 |
| 导出 | xlsx + html2canvas | Excel 导出 + 图片导出 |

### 4.2 AI 引擎（utils/ai.js — 10 个函数）

| 函数 | 调用场景 | 所属模块 |
|------|----------|----------|
| `aiAudit(ocrText)` | 标签 GB 7718 智能审核（7项规则） | AI智能 > 标签审核 |
| `aiExportAudit(ocrText, market)` | 按目标国标准审核标签 | AI智能 > 多国出口审核 |
| `aiNutritionGen(productName, ingredients)` | 根据配方计算营养成分（GB 28050） | AI智能 > 营养成分生成 |
| `aiIngredientsVerify(text)` | 配料表5维度校验 | AI智能 > 配料表校验 |
| `aiSupplierScore(data)` | 供应商4维度综合评分 | AI智能 > 供应商评分 |
| `aiComplaintClassify(text)` | 客诉自动分类+紧急定级 | AI智能 > 客诉分类定级 |
| `aiComplaintReply(info)` | 生成专业客诉回复话术 | AI智能 > 客诉回复生成 |
| `aiComplaintAnalysis(data)` | 客诉根因分析+整改建议 | AI智能 > 客诉根因分析 |

**降级策略：** 未配置 `AI_API_KEY` 时返回 null，Controller 层自动 fallback 到内置规则/模板。

## 六、安全措施

| 措施 | 说明 |
|------|------|
| 路由守卫 | 未登录自动跳转 `/login` |
| 密码加密 | SHA256 + 随机盐值 |
| XSS 防护 | 全局请求体 HTML 转义中间件 |
| 限流 | general (100/分钟) + strict (10/分钟 用于登录/注册/AI) |
| 文件上传 | 限制格式 + 大小限制 + multer 配置 |
| SQLite 外键 | FOREIGN KEY 约束保证数据完整性 |

## 九、启动方式

```bash
# 后端
cd server
npm install
node index.js          # http://localhost:3001

# 前端
cd client
npm install
npx vite               # http://localhost:3000（自动代理 /api → :3001）
```

**环境变量（.env.development）：**
```
PORT=3001
AI_API_KEY=          # DeepSeek API Key（不填则 AI 功能降级）
AI_BASE_URL=         # 默认 https://api.deepseek.com
```
